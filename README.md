Макет сделан мной  в процессе обучения верстке в Webacademy.ru. Дизайн (PSD) и верстка (HTML, LESS, jQuery).

    Мобильная навигация
    
    Глерея изображений fancybox
    
    Фильтр карточек в Портфолио
    
    Форма обратной связи (Ajax, PHP)
